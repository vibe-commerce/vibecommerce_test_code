## План построения модели

### Этап 1: Исследование рынка (уточнение параметров)

**1.1 Анализ конкурентов на МП**
- [ ] Найти топ-10 SPF-кремов премиум сегмента на Ozon/WB
- [ ] Выгрузить: цены, объёмы продаж, рейтинги, отзывы
- [ ] Определить реалистичный ценовой коридор (моя гипотеза: 1200-1800₽)

**1.2 Анализ сезонности**
- [ ] Построить график сезонности категории "солнцезащитные средства"
- [ ] Определить коэффициенты сезонности по месяцам (пик: май-август)

**1.3 Комиссии и логистика**
- [ ] Уточнить комиссию МП для категории "косметика/SPF" (ориентир: 15-20%)
- [ ] Рассчитать логистику FBO для разных объёмов упаковки

### Этап 3: Финансовая модель

**3.1 Структура P&L (помесячно, 36 месяцев)**

```
ДОХОДЫ
├── Выручка Ozon
├── Выручка Wildberries
├── Выручка собственный сайт
└── = ИТОГО Выручка

ПЕРЕМЕННЫЕ РАСХОДЫ
├── Себестоимость товара (COGS)
├── Комиссия Ozon (~18%)
├── Комиссия WB (~15%)
├── Эквайринг сайт (~2.5%)
├── Логистика FBO
├── Логистика до клиента (сайт)
├── Упаковка/маркировка
└── = ИТОГО Переменные

ВАЛОВАЯ ПРИБЫЛЬ = Выручка - Переменные

МАРКЕТИНГ
├── Внутренняя реклама МП
├── Внешний трафик (Telegram, блогеры)
├── Продвижение на сайте (Яндекс Директ)
└── = ИТОГО Маркетинг

ПОСТОЯННЫЕ РАСХОДЫ
├── Разработка бренда/упаковки (единоразово)
├── Сертификация продукции
├── Фотосъёмка/контент
├── Подписки (аналитика, сервисы МП)
├── ЗП/самозанятость
└── = ИТОГО Постоянные

EBITDA = Валовая прибыль - Маркетинг - Постоянные

НАКОПЛЕННЫЙ CASH FLOW
```

### Этап 5: Риски и сценарии

**5.1 Ключевые риски**
1. **Сезонность** — 70% продаж за 4 месяца, нужен кэш на межсезонье
2. **Конкуренция** — крупные бренды могут демпинговать
3. **Срок годности** — SPF обычно 2 года, риск списания остатков
4. **Сертификация** — задержки с документами могут сорвать сезон
