# Методология расчёта ценовой эластичности

## Эластичность спроса по цене (Price Elasticity of Demand)

### Формула — метод средней точки (Arc Elasticity)

```
E = (ΔQ / Q̄) / (ΔP / P̄)

где:
  ΔQ = Q₂ - Q₁         — изменение объёма продаж
  Q̄  = (Q₁ + Q₂) / 2   — средний объём
  ΔP = P₂ - P₁         — изменение цены
  P̄  = (P₁ + P₂) / 2   — средняя цена
```

Метод средней точки выбран потому, что даёт симметричный результат независимо от направления изменения цены.

### Интерпретация

| Значение E | Тип спроса | Что делать |
|------------|-----------|------------|
| E < -3 | Очень эластичный | Снижение цены резко увеличивает объём. Конкурировать ценой |
| -3 < E < -1 | Эластичный | Снижение цены увеличивает выручку. Акции эффективны |
| -1 < E < -0.5 | Пограничная зона | Тестировать осторожно |
| -0.5 < E < 0 | Неэластичный | Цена мало влияет на объём. Можно повышать |
| E > 0 | Товар Гиффена (редко) | Повышение цены увеличивает спрос (престиж/дефицит) |

### Типичные значения для e-commerce

- Бытовая электроника: E = -1.5 ... -3.0
- Одежда и обувь: E = -0.5 ... -1.5
- FMCG (еда, бытхим): E = -0.3 ... -0.8
- Премиум-товары: E = -0.1 ... -0.5
- Товары для сна (по данным репозитория): E = -0.2 ... -13.0

## Log-linear модель спроса

Для прогнозирования объёма при заданной цене используется модель:

```
Q = e^a × P^E

где:
  a — константа (калибруется по данным)
  E — эластичность (наклон в log-пространстве)
  P — цена
  Q — объём продаж
```

### Калибровка

Из двух наблюдаемых точек (P₁, Q₁) и (P₂, Q₂):

```
E_log = ln(Q₂/Q₁) / ln(P₂/P₁)
a     = ln(Q₁) - E_log × ln(P₁)
```

### Целевая цена для N× объёма

```
Q_target = N × Q_current
P_target = exp((ln(Q_target) - a) / E_log)
```

## Влияние на прибыль

При изменении цены обязательно считать:

```
Прибыль = (Цена - Себестоимость) × Объём

Прибыль_текущая  = (P_current - COGS) × Q_current
Прибыль_целевая  = (P_target - COGS) × Q_target
```

Снижение цены может увеличить выручку, но уменьшить прибыль — если маржа сжимается сильнее, чем растёт объём.

## Ограничения

1. **Экстраполяция**: модель калибрована на узком диапазоне цен (5-18%). При глубоких скидках эластичность может измениться
2. **Confounders**: сезонность, акции, конкуренты, позиция в выдаче влияют на продажи помимо цены
3. **Платформенный эффект**: разные площадки (WB/Ozon) имеют разную аудиторию и разную чувствительность к цене
4. **Временной лаг**: эффект от изменения цены может проявляться не мгновенно

## Рекомендации по тестированию

1. Снижать/повышать цену поэтапно: 5-10% за шаг
2. Замерять результат минимум 1-2 недели
3. Контролировать прочие факторы (рекламу, сезон)
4. Сравнивать с предыдущим аналогичным периодом
