# SEO для российского рынка

## Поисковые системы

| Поисковик | Доля трафика (РФ) | Приоритет |
|-----------|-------------------|-----------|
| Яндекс | ~55-60% | Основной |
| Google | ~35-40% | Важный |
| Mail.ru, Rambler | ~5% | Низкий |

Оптимизация нужна для ОБОИХ основных поисковиков.

## Яндекс: специфика индексации

### Отличия от Google
- **Индексация медленнее**: дни/недели vs часы/дни у Google
- **Поведенческие факторы** ценятся выше (время на сайте, CTR в выдаче, возвраты)
- **Региональность**: Яндекс привязывает сайт к региону (важно для локальных запросов)
- **ИКС** (Индекс качества сайта): аналог Domain Authority, виден в Вебмастере
- **Текстовые факторы**: Яндекс чувствительнее к переоптимизации (keyword stuffing)

### Яндекс.Вебмастер
- URL: webmaster.yandex.ru
- Верификация: HTML-файл `yandex_*.html` в корне ИЛИ meta-тег
- Проверить: индексация, ошибки, поисковые запросы, ИКС

### Яндекс-специфичные мета-теги
```html
<meta name="yandex-verification" content="{verification-code}">
```

### robots.txt для Яндекса
```
User-agent: Yandex
Allow: /
Disallow: /api/
Disallow: /course/checkout
Disallow: /course/success
Disallow: /course/fail

# Основное зеркало (Host устарел, но не вредит)
Host: https://domain.com

# Очистка UTM-параметров
Clean-param: utm_source&utm_medium&utm_campaign&utm_content&utm_term
```

## Исследование ключевых слов

### Яндекс Wordstat
- URL: wordstat.yandex.ru
- Бесплатный, требует Яндекс ID
- Форматы запросов:
  - `ключевое слово` — широкое совпадение (все формы)
  - `"ключевое слово"` — фразовое (в указанном порядке)
  - `!ключевое !слово` — точное (без склонений)
- Показывает: частотность по регионам, сезонность

### WebSearch подсказки для Wordstat
```
"wordstat {keyword}" — часто показывает частотность в результатах
"{keyword} частотность яндекс" — блогеры публикуют данные
```

### Альтернативы
- keys.so — платный, полный аналог Wordstat API
- Serpstat — частично поддерживает Россию
- Букварикс (bukvarix.com) — база ключевых слов Яндекса

## Яндекс.Метрика
- URL: metrika.yandex.ru
- ID счётчика в `<head>`: `<!-- Yandex.Metrika counter -->`
- WebVisor — уникальная функция записи сессий пользователей
- Проверить: карта кликов, вебвизор, конверсии

## Аналитика и мониторинг

### Yandex XML API (мониторинг позиций)
- URL: xml.yandex.ru
- Бесплатный лимит: 10-1000 запросов/сутки
- Требует: Яндекс ID + подтверждённый сайт в Вебмастере
- Возвращает: полную выдачу Яндекса для запроса
- Настройка: см. `seo-positions/references/tracking-setup.md`

### Сервисы мониторинга (платные)
- Topvisor (topvisor.com) — позиции в Яндексе и Google
- AllPositions (allpositions.ru) — массовая проверка
- SE Ranking — международный, поддерживает Яндекс

## AI-краулеры (AEO для РФ)
- **YandexGPT** (через Алису) — использует индекс Яндекса
- **Нейро** (Яндекс) — AI-ответы в поисковой выдаче
- Важно: НЕ блокировать в robots.txt GPTBot, PerplexityBot
- AEO пока менее развит в РФ, но быстро растёт

## Особенности контента для РФ
- Язык: русский, проверять через Яндекс.Спеллер
- Юридические страницы: оферта, политика конфиденциальности (152-ФЗ)
- Цены: в рублях (₽), актуальные
- Контакты: российский телефон, email, Telegram
