# On-Page SEO Checklist

## Heading Structure

### H1
- Exactly ONE per page
- Contains primary keyword
- Length: 20-70 characters
- Unique across the entire site

### H2-H6 Hierarchy
- Logical structure: H1 → H2 → H3 (no skips like H1 → H3)
- H2 for main sections (minimum 2-3 per content page)
- Contains relevant keywords naturally
- Descriptive (not generic like "More Info")

## Content Quality

### Minimum Word Count
| Page Type | Minimum | Recommended |
|-----------|---------|-------------|
| Homepage | 500 | 800-1500 |
| Product/Service page | 800 | 1500+ |
| Blog post / Article | 1500 | 2000-3000 |
| Technical guide | 2000 | 3000-5000 |
| Service page (checkout, etc.) | 100 | 300 |

### Keyword Placement
- Primary keyword in first 100 words
- Natural density: 1-2% (not more!)
- LSI keywords (semantically related terms) throughout
- Keywords in: H1, H2s, alt tags, first paragraph

### Readability
- Short paragraphs (2-4 sentences)
- Bullet and numbered lists
- Visual content breaks (images, tables, quotes)
- Simple language (explain jargon when used)

### Freshness
- Publication date visible
- Last update date visible (if updated)
- Information is current and accurate
- No dead external links

## Internal Linking
- **3-5+ internal links** per page to relevant pages
- **Descriptive anchor text** (not "click here", "read more")
- Links are **contextually relevant**
- Logical site structure through links
- **Breadcrumbs** present for navigation

## External Links
- Links to **authoritative sources**
- `rel="nofollow"` for affiliate/advertising links
- `rel="noopener"` for `target="_blank"` links
- `rel="sponsored"` for paid links

## Image SEO
### Alt Text
- Descriptive, contains relevant keywords
- Unique per image
- Max 125 characters
- Empty `alt=""` only for decorative images

### File Names
- Descriptive: `red-leather-sofa.jpg` (not `IMG_1234.jpg`)
- Use hyphens between words
- Lowercase

## Above the Fold
- Value proposition visible immediately
- Primary CTA noticeable and clear
- Content loads < 1.5s
- No intrusive popups on entry (Google penalizes this)

## Content Formatting
- **Bold** for important phrases (with keywords)
- Tables for structured comparisons
- Blockquotes for testimonials/citations
- Numbered lists for sequences
- Bullet lists for features/benefits

## Multimedia
- Video with transcript on page (if applicable)
- Schema VideoObject markup for videos
- Infographics for data visualization
- All media has context (captions, descriptions)

## User Experience
- Clear navigation (header + footer)
- Mobile menu works properly
- Custom 404 page with navigation
- Contact information easy to find
- CTA buttons with contrasting colors
- Minimal form fields
- Social proof elements (reviews, ratings, logos)

## Accessibility
- Sufficient text contrast (WCAG AA: 4.5:1 ratio)
- Keyboard navigation works
- Focus indicators visible
- ARIA labels for interactive elements
- Semantic HTML used correctly
