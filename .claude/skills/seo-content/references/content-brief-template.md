# Content Brief Template

Use this format when generating content briefs.

```markdown
# Content Brief: {Topic Title}

## Target Keywords

### Primary Keyword
**{keyword}** — Volume: {N or HIGH/MEDIUM/LOW} — Difficulty: {N or estimate}

### Supporting Keywords
| Keyword | Volume | Intent |
|---------|--------|--------|
| {kw1} | ... | informational |
| {kw2} | ... | commercial |
| {kw3} | ... | transactional |

## Content Requirements

| Parameter | Requirement |
|-----------|-------------|
| **Format** | {guide / listicle / comparison / tutorial / review} |
| **Word count** | {N-N words} (SERP average: {N}) |
| **Reading level** | {general audience / professional / technical} |
| **Language** | {ru / en / other} |
| **Schema.org** | {Article / TechArticle / HowTo / FAQPage} |

## SEO Requirements

- [ ] Primary keyword in title (50-60 chars)
- [ ] Primary keyword in H1 (20-70 chars)
- [ ] Primary keyword in first 100 words
- [ ] Supporting keywords in H2 headings
- [ ] Meta description with CTA (150-160 chars)
- [ ] 3-5+ internal links to relevant pages
- [ ] Alt text on all images with keywords
- [ ] Canonical URL set

## AEO Requirements

- [ ] Each paragraph understandable without context (autonomous fragments)
- [ ] Semantic HTML: use <dl>, <figure>, <blockquote>, <details> where appropriate
- [ ] Consistent terminology throughout (define abbreviations once)
- [ ] datePublished and dateModified in Schema
- [ ] Author info with credentials in Schema
- [ ] FAQ section with <details>/<summary> markup

## Recommended Outline

### H1: {Primary keyword + value proposition}

### H2: {Section 1 — addresses primary intent}
- Key points to cover: ...
- Supporting keywords to include: {kw1, kw2}

### H2: {Section 2}
- Key points: ...
- Include: {table / list / comparison}

### H2: {Section 3}
- Key points: ...

### H2: FAQ
- {Question 1 from "People Also Ask"}?
- {Question 2}?
- {Question 3}?

## SERP Competitor Analysis

### What top competitors do well
- {Pattern 1}
- {Pattern 2}

### Content gaps (your opportunity)
- {Gap 1 — none of top 5 cover this}
- {Gap 2}

### Recommended differentiation angle
{How to stand out from existing content}

## Internal Linking Plan
- Link TO this page from: {existing page 1}, {existing page 2}
- Link FROM this page to: {related page 1}, {related page 2}

## Suggested Title Variants
1. "{Variant 1}" ({N chars})
2. "{Variant 2}" ({N chars})
3. "{Variant 3}" ({N chars})

## Suggested Meta Description
"{Description with primary keyword and CTA}" ({N chars})
```
