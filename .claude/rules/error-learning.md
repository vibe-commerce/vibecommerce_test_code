# Error Learning

После бага, упавшего скрипта или 2+ неудачных попыток:

1. Запиши в `.claude/data/error-log.md`:
   - Симптом (что случилось)
   - Причина (почему)
   - Исправление (что сделано)
   - Предотвращение (как не повторить)

2. Обнови инструкции, если ошибка выявила пробел в правилах.

3. Правило трёх: кодифицируй правило сразу, только если фикс очевиден.
   Иначе жди 3 повторения одного класса ошибки.
