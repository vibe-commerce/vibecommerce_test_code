# Бэклог задач

Last Updated: 2026-02-12

## Описание
Единое место хранения всех задач по всем проектам.

## Ключевые файлы
- `backlog.xlsx` — Single Source of Truth (все задачи) [CANONICAL]
- [new-tasks.md](new-tasks.md) — входящие задачи (свободный формат, для быстрого добавления)

## Структура backlog.xlsx
| Колонка | Описание |
|---------|----------|
| n | Уникальный ID |
| PRJ | Код проекта |
| Название задачи | Action-oriented заголовок |
| Описание задачи | Что сделать |
| Приоритет | 1–100 |
| Дедлайн | YYYY-MM-DD |
| Спринт | Дата окончания спринта |
| Статус | TO DO / WIP / ON PRIORITY / DONE / ON HOLD |
| Комментарий | Последний апдейт |

## Workflow
1. Новые задачи → `new-tasks.md`
2. `/project-manager` переносит в `backlog.xlsx`
3. Приоритизация через `/project-manager приоритизация`
4. Работа по спринтам
