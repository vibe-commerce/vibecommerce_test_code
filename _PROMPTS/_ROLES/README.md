# Экспертные роли

Last Updated: 2026-02-12

## Описание
Роли задают глубокий экспертный контекст для Claude Code. Загружаются автоматически по теме запроса.

## Доступные роли

| Файл | Роль | Когда используется |
|------|------|--------------------|
| [Head_of_E-Commerce.md](Head_of_E-Commerce.md) | Head of E-Commerce | E-commerce вопросы, карточки, unit-экономика, маркетплейсы |
| [Project_Manager.md](Project_Manager.md) | Project Manager | Планирование, бэклог, спринты, статус-отчёты |

## Как добавить роль
1. Создай файл `Название_Роли.md` в этой папке
2. Используй шаблон из [repo-setup-guide-ecom.md](../../repo-setup-guide-ecom.md#9-роли)
3. Добавь роль в секцию `Role Selection` в `CLAUDE.md`
4. Обнови эту таблицу
